GAME CONTROLS

W - Pitch Up 
S - Pitch Down
A - Turn Left
D - Turn Right 
Q - Roll Left
E - Roll Right

SPACE - Thrust

Left Mouse Button - Fire Missiles 

1,2,3,4 - Change Camera POV 

ESC - Pause
